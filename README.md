# Ekasutram-website
Steps to host the website

1. Frontend(React + Vite + TypeScript)
  Just select the frontend folder from the directory and add VITE_API_URL(Backend URL) in environment variables as Your  URL + /api

2.Backend(Java(SpringBoot))
  Just select the backend folder from the directory and add 
  ADMIN_PASSWORD, FRONTEND_URL(Generated after deploying frontend), SUPABASE_BUCKET, SUPABASE_KEY, SUPABASE_URL
  in environment variables.


  Host any of the one first then accordingly update the urls.
